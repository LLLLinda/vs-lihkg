# vs-lihkg README

This is for browsing LIHKG in vs-code when you in office. 

## How to install

1. download the project and find vs-lihkg-0.0.1.vsix in root folder.
2. Open the Extensions tab and click "install from VSIX" and select above file
3. enjoy!
